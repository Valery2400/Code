Создать набор классов и методов, которые будут:
1) добавлять в массив сортов чая разные элементы
2) сортировать чай по параметру, который выберет пользователь
3) найти среди чая нужный сорт - есть ли он или нет

===================================


Для решения нам необходимо:
1) класс "Производитель" с полем "наименование"



2) класс "Чай" с полями:
    - производитель - экземпляр класса "Производитель"
    - сорт - String

3) класс "Все сорта чая" - где будут хранится массив "Все чаи"
и в этом классе будут все необходимые методы для работы с этим массивом

ОПРЕЦИИ С МАММИВАМИ (СТРУКТУРАМИ ДАННЫХ)

- read (чтение, чаще задача поиска элемента в массиве)
- update (обновление данных)
- delete (удаление данных)
