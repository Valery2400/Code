Написать приложение, которое позволяло бы учитывать книги в библиотеке

Требования к приложению:
Хранение книг должно осуществляться следующим образом:
- иметь возможность учитывать отдельную книгу
- книга хранится на полке с номером
- полка находится в определенном шкафу

Количество полок в каждом шкафу одинаково

поиск:
- по автору
- по названию
